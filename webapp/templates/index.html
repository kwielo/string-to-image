String to image API

Usage

Parameters for /api/v1/string-to-image or /string-to-image.png

  • text
  • font_name
  • font_size

Parameters for /api/v1/json-to-image or /json-to-image.png

  • json_url
  • json_path
  • font_name
  • font_size

Examples

http://localhost:8080/api/v1/string-to-image?text=2.12.2-rc.10&font_size=40

http://localhost:8080/api/v1/json-to-image?json_url=https://example.com/version.json&json_path=version
